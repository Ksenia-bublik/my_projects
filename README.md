# my_projects
## 1) Проект по эконометрике: анализ гендерного разрыва в заработных платах

В данном проекте исследуется наличие и величина гендерного разрыва в заработных платах на микроданных по России. В работе проводится подготовка и очистка данных, построение эконометрических моделей, проверка статистических гипотез и интерпретация полученных результатов. Отдельное внимание уделено сравнению альтернативных спецификаций модели и диагностике их корректности.

В репозитории представлены основные материалы проекта: код исследования, текст работы и презентация. Исходный датасет с сырыми данными не включён в репозиторий, поскольку его объём превышает допустимый для загрузки размер. Тем не менее, все ключевые результаты, логика анализа и структура исследования сохранены в приложенных файлах.

## 2) Banking Data Analysis (Excel + Power Query)

В рамках тестового задания была выполнена обработка и базовый аналитический разбор банковского датасета в Excel с использованием Power Query. Основная цель работы состояла не только в очистке и структурировании данных, но и в подготовке понятных итоговых таблиц, позволяющих быстро получить общее представление о клиентской базе, транзакционной активности и кредитных продуктах.

В процессе работы исходный датасет был загружен в Power Query, после чего были выполнены преобразования, необходимые для дальнейшего анализа: проверка структуры данных, выделение клиентского среза, создание производных признаков, а также расчёт показателей, связанных с движением денежных средств. Для транзакций был рассчитан показатель Net Cash Flow, дополнительно были выделены признаки года, месяца и типов операций, что позволило построить сводки по динамике и структуре денежных потоков.

На основе подготовленных данных были собраны итоговые аналитические листы. В частности, были сформированы:

общий Summary с ключевыми KPI;
сводка по городам (City Summary);
сегментация по полу и типу счёта (Gender & Account Type Summary);
анализ транзакций по датам, типам операций и клиентам;
отдельная сводка по кредитам (Loan Summary) с количеством заявок, статусами, approval rate, средней суммой кредита, средней ставкой и сроком;
отдельная сводка по кредитным картам (Card Summary) со средним лимитом, средним балансом, средней утилизацией лимита и долей клиентов с высокой кредитной нагрузкой.

Работа была выполнена в Excel без использования Python или SQL: вся логика подготовки данных, агрегации и расчётов реализована средствами Power Query, итоговые результаты выгружены в удобный для просмотра формат внутри одного Excel-файла.

Примечание: датасет содержит данные по городам США, поэтому денежные показатели интерпретируются как суммы в USD.
